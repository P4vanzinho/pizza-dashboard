# Pizza Shop Dashboard

A modern and intuitive dashboard for pizzeria management, built with React, TypeScript, and Tailwind CSS.

## Features

- Real-time Dashboard Metrics
  - Daily/Monthly Revenue
  - Total Orders
  - Cancellation Rate
  - Interactive Sales Charts

- Order Management
  - Advanced Filter Listing
  - Order Details
  - Status Updates
  - Complete History

- Design System
  - Light/Dark Theme
  - Reusable Components
  - Consistent & Responsive UI
  - Accessibility

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Shadcn/ui
- React Router
- React Query
- React Hook Form + Zod
- Recharts
- Lucide Icons

## Getting Started

1. Clone the repository
git clone https://github.com/yourusername/pizza-shop-dashboard.git

2. Install dependencies
npm install

3. Run the development server
npm run dev
